# 🐔 Market Price - Android App

Welcome to **Market Price**, your modern companion for tracking real-time poultry and market trends. Built for speed, reliability, and a premium user experience.

## 📥 Download and Install
Since we are in open-beta, you can download the application directly from GitHub for **FREE**.

### **Step 1: Download the App**
Click the link below to download the latest **APK** file:
👉 **[Download Market Price v1.0.0 (APK)](https://github.com/PVenkataNarasimha/Price/releases)**

### **Step 2: Install on Android**
1.  Open the downloaded `.apk` file on your phone.
2.  Your phone may ask: *"Install from unknown sources?"* 
3.  Click **Settings** -> **Allow from this source**.
4.  Return and click **Install**.

---

## ✨ Features
- **Real-time Pricing:** Stay updated with the latest chicken and broiler market rates.
- **Admin Dashboard:** Powerful tools for market managers to update prices instantly.
- **Modern Design:** A clean, red-themed interface designed for clarity and speed.
- **Offline Ready:** View the last checked prices even when you're not connected.

## 🛠️ Tech Stack
- **Frontend:** React Native (Expo)
- **Backend:** Node.js & Express (Hosted on Render)
- **Database:** MongoDB Atlas

---

## 👨‍💻 Development
If you want to run the project locally:

1.  **Clone the repo:**
    ```bash
    git clone https://github.com/PVenkataNarasimha/Price.git
    ```
2.  **Install dependencies:**
    ```bash
    cd mobile
    npm install
    ```
3.  **Start the app:**
    ```bash
    npx expo start
    ```
